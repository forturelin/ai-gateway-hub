关于

v1.3.0

核心功能

  • 多供应商汇聚：统一管理 OpenAI / Anthropic 兼容协议的 API 供应商
  • 协议全矩阵转换：OpenAI ↔ Anthropic 双向自动转换，任意客户端对接任意供应商
  • 灵活映射规则：自定义模型别名、多规则转发、5 种切换策略（固定 / 顺序 / 随机 / 最少使用 / 时间段轮换）
  • 费用追踪：内置 17 款模型定价，支持自定义价格，实时计算每次请求费用
  • 数据看板：今日 / 本月统计、模型分布、供应商占比、调用趋势
  • 用量分析：按时间范围（1 天 ~ 12 个月）查看请求数、Token、费用、错误率
  • 请求日志：365 天完整记录，支持搜索、筛选、详情查看、JSON / CSV 导出
  • 聊天测试：页面内直接调试对话，支持流式 / 非流式，可调节温度参数
  • 7 套主题：浅色 / 深色 / 跟随系统 / 海蓝 / 森绿 / 暮紫 / 暖纸
  • 后台运行：交互式控制面板，一键启动 / 停止 / 重启，跨平台支持
